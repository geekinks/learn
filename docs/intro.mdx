# 🌍 **Geek Ink – 2-Week Web Development Introduction**  
> 🚀 **Goal:** Equip students with foundational web development knowledge before diving into specialized frontend or backend tracks.  

## 🎯 **Program Overview**  
✅ **Duration:** 2 Weeks (4 Sessions)  
✅ **Meeting Frequency:** Twice per week  
✅ **Tech Stack:**  
   - 🖥 **Frontend:** HTML, CSS, JavaScript  
   - 🌐 **Backend:** Intro to Servers, APIs, Databases  
   - 🛠 **Tools:** Git & GitHub, VS Code  

---

<details>
<summary >📌 Week 1: Introduction to the Web & Frontend Basics</summary>

### **Session 1: How the Web Works & HTML Basics**  
✔ **Topics:**  
   - How the internet works (**Clients, Servers, HTTP, DNS**)  
   - Introduction to **HTML** (Headings, Links, Forms)  

✔ **Practical Activities:**  
   - Set up **VS Code & GitHub Pages**  
   - Create a **personal profile page**  

✔ **Deliverable:**  
   - A **hosted profile webpage**  

📖 **Learning Resources:**  
   - [🔗 MDN HTML Basics](https://developer.mozilla.org/en-US/docs/Learn/HTML)  
</details>

<details>
<summary >🎨 Session 2: Styling with CSS</summary>

✔ **Topics:**  
   - CSS Selectors, Colors, Fonts  
   - Responsive Design (**Media Queries, Flexbox**)  

✔ **Practical Activities:**  
   - Style the **profile page**  
   - Implement **a responsive layout using Flexbox**  

✔ **Deliverable:**  
   - A **fully styled, responsive webpage**  

📖 **Learning Resources:**  
   - [🔗 CSS Tricks – Flexbox Guide](https://css-tricks.com/snippets/css/a-guide-to-flexbox/)  
</details>

---

<details>
<summary >📌 Week 2: JavaScript Basics & Intro to Backend</summary>

### **Session 3: JavaScript for Beginners**  
✔ **Topics:**  
   - JavaScript Variables & Data Types  
   - Functions, Events & DOM Manipulation  

✔ **Practical Activities:**  
   - Add **interactive elements (clickable buttons, dynamic text updates)**  
   - Build a **simple To-Do List app**  

✔ **Deliverable:**  
   - A **JavaScript-powered interactive webpage**  

📖 **Learning Resources:**  
   - [🔗 MDN JavaScript Guide](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Guide)  
</details>

<details>
<summary >🖥️ Session 4: Introduction to Backend & Databases</summary>

✔ **Topics:**  
   - What are **Servers & APIs**?  
   - Basics of **Node.js, Express & MongoDB**  

✔ **Practical Activities:**  
   - Make a **simple API call using Fetch**  
   - Discuss **backend’s role in handling data**  

✔ **Deliverable:**  
   - A **working API request demo**  

📖 **Learning Resources:**  
   - [🔗 What is an API?](https://developer.mozilla.org/en-US/docs/Learn/JavaScript/Client-side_web_APIs/Introduction)  
</details>

## 🚀 **Final Task: Choose Your Track**  
By the end of these two weeks, students will:

✅ Be ready to join:  
   - 🎨 **Frontend Track** (**React.js, UI/UX, API Integration**)  
   - 🖥 **Backend Track** (**Node.js, Express, MongoDB**)
